show-file-on-disk

Eclipse plugin which allows you to navigate (via Windows Explorer/Finder) to file on the disk

Download this project as a .zip file Download this project as a tar.gz file

What is it?

The Eclipse plug-in which allows the users to navigate to actual file on the operating system which is mapped to the selected Eclipse resource in the Eclipse views like Project Explorer, Package Explorer, etc.

Installation instructions

a) Add update site

Update site: http://show-file-on-disk.googlecode.com/svn/trunk/site/v1/

b) Select the feature "Show File on Disk

Usage

Now a new menu item will be available on the context menu's in Project Explorer, Package Explorer, Search view, etc.

Screen showing new menu item "Show in Windows Explorer" (For OSX it will be "Show in Finder").
